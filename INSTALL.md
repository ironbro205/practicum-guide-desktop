# 실습가이드 데스크톱 앱 — 설치 안내 (선생님용)

실습가이드 사이트를 컴퓨터 프로그램처럼 쓸 수 있게 해 주는 앱입니다.
설치하면 **새 공지·채널 글이 올라올 때 화면 오른쪽 아래에 알림**이 떠요.

---

## 1. 내려받기

1. 아래 주소를 열면 **PracticumGuide-Setup.exe** 파일이 바로 내려받아져요.
   (항상 최신 버전이 받아지는 고정 주소입니다.)
   https://github.com/ironbro205/practicum-guide-desktop/releases/latest/download/PracticumGuide-Setup.exe
2. 실습가이드 **홈 화면 오른쪽 아래 [앱 다운로드] 버튼**으로도 받을 수 있어요.

## 2. 파란 경고창이 떠도 정상이에요

실행하면 "Windows의 PC 보호" 라는 파란 창이 뜰 수 있어요.
학교 밖에서 만든 프로그램이라 뜨는 안내일 뿐, 문제가 있는 게 아닙니다.

1. 파란 창에서 **"추가 정보"** 를 눌러 주세요. (작은 글씨예요)
2. 그러면 나타나는 **"실행"** 단추를 눌러 주세요.

*(여기에 스크린샷: 파란 경고창 → 추가 정보 → 실행)*

## 3. 설치는 클릭 한 번

- 실행하면 **묻는 것 없이 자동으로 설치**되고 바로 앱이 켜집니다.
- **관리자 암호는 필요 없어요.** (내 계정 폴더에만 설치됩니다)
- 처음 한 번만 실습가이드에 로그인하면, 그다음부터는 로그인이 유지돼요.

## 4. 사용법 — 꼭 알아둘 것 하나

**오른쪽 위 X 를 눌러도 앱이 꺼진 게 아닙니다.**
화면 오른쪽 아래 **시계 옆 작은 아이콘**으로 숨어서 계속 알림을 지켜봐요.

- 다시 열기: 시계 옆 실습가이드 아이콘을 **한 번 클릭**
- 완전히 끄기: 아이콘에 **마우스 오른쪽 클릭 → 종료**
- 컴퓨터를 켜면 앱이 조용히 자동으로 시작돼요.
  원치 않으면: 아이콘 오른쪽 클릭 → **"부팅 시 자동실행"** 체크를 꺼 주세요.

## 5. 알림은 이렇게 와요

- 새 공지나 채널 글이 올라오면 **1분 안에** 알림이 떠요.
- 알림을 클릭하면 해당 화면이 바로 열립니다.
- 새 알림이 오면 **시계 옆 실습가이드 아이콘이 빨간 점과 함께 깜빡여요.**
  깜빡임은 앱 창을 한 번 열면 사라집니다.
- 로그인이 풀리면 "다시 로그인해 주세요" 알림이 한 번 떠요 —
  앱을 열어 다시 로그인하면 됩니다.

## 6. 업데이트는 자동

새 버전이 나오면 **알아서 내려받아 설치**됩니다. 아무것도 안 하셔도 돼요.
(경고창 없이 조용히 진행됩니다.)

## 7. 자주 묻는 것

**Q. 백신(V3 등)이 "시작 프로그램에 등록됐다"고 알려요.**
정상이에요. 이 앱이 부팅 시 자동실행되도록 등록한 것입니다. "허용"을 눌러 주세요.

**Q. 알림이 안 와요.**
① 시계 옆 아이콘이 있는지(앱이 켜져 있는지) ② 앱에서 로그인이 되어 있는지
순서로 확인해 주세요. (Windows 알림 설정과는 관계없어요 — 아래 8번 참고)

**Q. 지우고 싶어요.**
Windows 설정 > 앱 > 설치된 앱에서 "실습가이드"를 찾아 제거하면 됩니다.
제거 후 작업 관리자 > 시작 앱 목록에 "실습가이드"가 남아 있으면
(실행될 파일이 없어 해는 없지만) 그 항목에서 "사용 안 함"으로 정리해 주세요.

## 8. 문제 해결 — 알림 배너

이 앱의 알림은 **앱이 직접 화면 오른쪽 아래에 띄우는 자체 배너**예요.
**Windows 알림 설정(방해 금지, 배너 끔 등)과 관계없이 항상 뜹니다** —
따로 설정을 만질 필요가 없어요.

- **잘 뜨는지 확인**: 시계 옆 실습가이드 아이콘에
  **마우스 오른쪽 클릭 → "알림 테스트"** 를 누르면 즉시 배너가 하나 떠요.
- 배너는 **7초 뒤 저절로 사라지고**, 클릭하면 해당 화면이 열려요.
  × 를 누르면 그냥 닫혀요.
- 배너를 놓쳐도 걱정 없어요 — 새 알림이 오면
  **시계 옆 아이콘이 빨간 점과 함께 깜빡이니까**, 깜빡이면 앱을 열어 보면 됩니다.

참고: "방해 금지를 꺼 주세요" 같은 Windows 알림 설정 안내는
**옛 버전(1.0.1 이하)에만 해당**하고, 지금 버전에서는 필요 없습니다.
